Naming Elements

div tags - No semantic meaning, div is like an empty box you place content in to assist in styling or adding JavaScript effects.

ID - id="unique-name" Each id should be unique.
Class - class="give-name" Classes can be assigned to multiple elements. Recommended for styling and microformats.
